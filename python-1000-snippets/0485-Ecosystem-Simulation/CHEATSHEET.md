# 0485-Ecosystem-Simulation Cheatsheet

## Quick Start
Run a sample:
```bash
python3 python-1000-snippets/0485-Ecosystem-Simulation/SAMPLES/sample1.py
```

## Notes
- `sample1.py`: Lotka-Volterra model over 20 iterations.
- `sample2.py`: Exports prey/predator time series to `temp/ecosystem.csv`.
- `sample3.py`: Evaluates growth versus decline conditions.

## Tips
- Tune constants to test equilibrium vs oscillation.
- Add random perturbations to model more complex ecological behavior.
