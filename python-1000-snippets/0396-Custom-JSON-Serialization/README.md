
# Custom JSON Serialization

## Description
Demonstrates how to serialize custom Python objects to JSON.

## Files
- `SAMPLES/sample1.py` — Use a `to_json` helper method.
- `SAMPLES/sample2.py` — Use `json.dumps(..., default=...)`.
- `SAMPLES/sample3.py` — Use `dataclasses` and `asdict`.

## Usage
```bash
python SAMPLES/sample1.py
python SAMPLES/sample2.py
python SAMPLES/sample3.py
```
