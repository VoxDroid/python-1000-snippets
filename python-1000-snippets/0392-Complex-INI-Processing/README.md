
# Complex INI Processing

## Description
Demonstrates parsing and writing INI configuration files using `configparser`.

## Files
- `SAMPLES/sample1.py` — Parse an INI file with sections.
- `SAMPLES/sample2.py` — Write an INI file programmatically.
- `SAMPLES/sample3.py` — Use interpolation across sections.

## Usage
```bash
python SAMPLES/sample1.py
python SAMPLES/sample2.py
python SAMPLES/sample3.py
```
