# User Behavior Analytics

## Description
Analyzes user event streams to compute actionable metrics, with or without pandas.

## Samples
- `SAMPLES/sample1.py`: count clicks from events.
- `SAMPLES/sample2.py`: calculate per-user click rates.
- `SAMPLES/sample3.py`: persist summary to `temp/0550_user_behavior.txt`.

## Output
- click counts and click rates.
- file entry with summary in temp.

## Explanation
User behavior analytics informs product decisions; compute event aggregates and percentiles.
