# Demand Forecasting

## Description
Forecasts future demand with linear trend from past data.

## Samples
- `SAMPLES/sample1.py`: fit linear model and forecast next period.
- `SAMPLES/sample2.py`: evaluate mean absolute error on history.
- `SAMPLES/sample3.py`: write forecast output to `temp/0565_forecast.txt`.

## Output
- projected value and metrics.
- temp output file.
