# K-Means Clustering

## Description
This snippet demonstrates K-Means clustering using `scikit-learn` to group synthetic data.

## Code
In the `SAMPLES/` folder you will find three examples:

- `sample1.py` — cluster synthetic data and print cluster centers/labels.
- `sample2.py` — compute inertia for varying `k` (elbow method).
- `sample3.py` — predict cluster assignment for new points.

Run any of them with:

```bash
python python-1000-snippets/0251-K-Means-Clustering/SAMPLES/sample1.py
```

## Output
Each sample prints clustering results and demonstrates how to use scikit-learn's `KMeans` API.

## Explanation
- **K-Means Clustering**: Iteratively assigns points to the nearest cluster center and updates centers.
- **Logic**: Uses synthetic data generated by `make_blobs`, fits K-Means, and inspects results.
- **Use Case**: Useful for unsupervised grouping, segmentation, and preprocessing.
- **Best Practice**: Scale features; choose `k` using inertia/elbow method; run multiple initializations (`n_init`).
