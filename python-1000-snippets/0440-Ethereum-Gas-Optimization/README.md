# Ethereum Gas Optimization

## Description
This snippet demonstrates using `web3` and `eth-tester` to estimate and compare gas usage when deploying and interacting with smart contracts.

## Requirements
- Python 3.8+
- `web3` (`pip install web3`)
- `eth-tester` (`pip install eth-tester`)
- `py-solc-x` (`pip install py-solc-x`)

## Samples
- `SAMPLES/sample1.py`: Estimate and measure gas for contract deployment.
- `SAMPLES/sample2.py`: Estimate and measure gas for a state-changing function call.
- `SAMPLES/sample3.py`: Compare deployment gas with and without Solidity compiler optimizations.

## Running
```bash
python python-1000-snippets/0440-Ethereum-Gas-Optimization/SAMPLES/sample1.py
python python-1000-snippets/0440-Ethereum-Gas-Optimization/SAMPLES/sample2.py
python python-1000-snippets/0440-Ethereum-Gas-Optimization/SAMPLES/sample3.py
```
