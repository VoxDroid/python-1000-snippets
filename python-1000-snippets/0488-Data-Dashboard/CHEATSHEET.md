# 0488-Data Dashboard Cheatsheet

## Quick Start
Run a sample:
```bash
python3 python-1000-snippets/0488-Data-Dashboard/SAMPLES/sample1.py
```

## Tips
- `sample1.py` prints a simple table to the console.
- `sample2.py` prints summary statistics (min/max/mean).
- `sample3.py` writes a dashboard report to `temp/dashboard.txt` in a loop.
- Adjust the dataset and formatting functions to display your own metrics.
