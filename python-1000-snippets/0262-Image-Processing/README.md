# Image Processing

## Description
This snippet demonstrates basic image processing using Pillow (PIL) and NumPy.

## Code
The `SAMPLES/` folder includes:

- `sample1.py` — creates a simple image with text and saves it.
- `sample2.py` — resizes, rotates, and crops an image.
- `sample3.py` — converts an image to grayscale and computes a histogram.

Run a sample with:

```bash
python python-1000-snippets/0262-Image-Processing/SAMPLES/sample1.py
```

## Output
Each sample generates one or more image files and prints status messages.

## Notes
- This snippet uses Pillow (PIL) for image creation and transformations.
- Use NumPy to analyze image pixel arrays (e.g., histograms).
