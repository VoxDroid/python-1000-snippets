# 0168-Adapter-Pattern Cheatsheet

- **Purpose**: convert one interface to another so that incompatible classes can work together.
- **Types**: object adapter (wraps instance) or class adapter (uses inheritance).
- **Technique**: implement target interface and delegate calls to the adaptee.
- **Use cases**: integrate legacy/third-party code without modifying it.
- **Python tip**: duck typing often reduces need for adapters; use when interfaces differ significantly.
