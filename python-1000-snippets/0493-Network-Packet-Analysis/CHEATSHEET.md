# 0493-Network Packet Analysis Cheatsheet

## Quick Start
Run a sample:
```bash
python3 python-1000-snippets/0493-Network-Packet-Analysis/SAMPLES/sample1.py
```

## Notes
- `sample1.py`: parse plain text packets and count TCP.
- `sample2.py`: parse hex packet bytes and print size/header.
- `sample3.py`: detect SYN-flood signature.

## Tips
- Convert this into CSV or JSON for real packet logs.
