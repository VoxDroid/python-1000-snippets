# 0499-Log-Analysis Cheatsheet

## Quick Start
```bash
python3 python-1000-snippets/0499-Log-Analysis/SAMPLES/sample1.py
```

## Notes
- `sample1.py`: extracts event names from log lines.
- `sample2.py`: counts events by type.
- `sample3.py`: writes filtered login events to `temp/log_filter.txt`.

## Tips
- Keep your log parser resilient to malformed lines.
- Use `temp/` for output to match repository structure.
