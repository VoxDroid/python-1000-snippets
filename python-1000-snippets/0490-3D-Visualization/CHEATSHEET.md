# 0490-3D-Visualization Cheatsheet

## Quick Start
Run a sample:
```bash
python3 python-1000-snippets/0490-3D-Visualization/SAMPLES/sample1.py
```

## Notes
- `sample1.py` writes a PLY file (`temp/cube.ply`) containing 3D point cloud vertices.
- `sample2.py` writes an OBJ file (`temp/heightmap.obj`) representing a simple heightmap mesh.
- `sample3.py` prints mesh statistics (centroid and bounds) for a simple mesh.

## Tips
- Open the generated `.ply` and `.obj` files in tools like MeshLab, Blender, or CloudCompare.
- Modify the `generate_*` functions to create different shapes and meshes.
