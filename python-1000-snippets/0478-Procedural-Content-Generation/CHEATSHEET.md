# 0478-Procedural-Content-Generation Cheatsheet

## Quick Tips
- Cellular automata generate organic structures by repeatedly applying simple local rules.
- Randomized Prim's algorithm produces perfect mazes (one unique path between any two points).
- Layering smoothed noise at different scales yields more natural heightmaps.

## Running examples
- `python SAMPLES/sample1.py` — dungeon generation via cellular automata.
- `python SAMPLES/sample2.py` — maze generation using randomized Prim.
- `python SAMPLES/sample3.py` — generate a heightmap using layered noise.
