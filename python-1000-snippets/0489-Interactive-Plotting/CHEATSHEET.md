# 0489-Interactive Plotting Cheatsheet

## Quick Start
Run a sample:
```bash
python3 python-1000-snippets/0489-Interactive-Plotting/SAMPLES/sample1.py
```

## Notes
- This snippet generates plot data and configuration files, suitable for use with interactive plotting tools.
- `sample1.py` writes CSV data to `temp/interactive_plot_data.csv`.
- `sample2.py` prints a simple dataset to the console.
- `sample3.py` writes a JSON plot configuration to `temp/interactive_plot_config.json`.

## Tip
After generating the data, load it into a tool like Plotly, Altair, or a spreadsheet to interactively explore it.
