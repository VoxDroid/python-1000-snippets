
# Automated Database Migration

## Description
Demonstrates a simple database migration workflow using SQLAlchemy.

## Files
- `SAMPLES/sample1.py` — Create a table and insert/query.
- `SAMPLES/sample2.py` — Add a column to an existing table (SQLite).
- `SAMPLES/sample3.py` — Show check-first table creation and reflection.

## Usage
```bash
python SAMPLES/sample1.py
python SAMPLES/sample2.py
python SAMPLES/sample3.py
```
