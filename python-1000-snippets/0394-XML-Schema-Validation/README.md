
# XML Schema Validation

## Description
Demonstrates validating XML against an XSD schema using `lxml`.

## Files
- `SAMPLES/sample1.py` — Validate a valid XML document.
- `SAMPLES/sample2.py` — Validate an invalid XML document and display the error.
- `SAMPLES/sample3.py` — Show compiling an XSD and reusing it for multiple documents.

## Usage
```bash
python SAMPLES/sample1.py
python SAMPLES/sample2.py
python SAMPLES/sample3.py
```
