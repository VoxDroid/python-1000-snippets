# 0491-Heatmap Generation Cheatsheet

## Quick Start
Run a sample:
```bash
python3 python-1000-snippets/0491-Heatmap-Generation/SAMPLES/sample1.py
```

## Notes
- `sample1.py` saves heatmap values to `temp/heatmap.csv`.
- `sample2.py` prints basic statistics (min/max/mean) for the heatmap.
- `sample3.py` renders a simple ASCII heatmap in the terminal.

## Tips
- Adjust the matrix size and value generation in `sample1.py` to explore different patterns.
- Use the CSV output for plotting with external tools if desired.
