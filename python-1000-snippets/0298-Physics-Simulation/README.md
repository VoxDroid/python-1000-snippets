# Physics Simulation

## Description
This snippet demonstrates basic physics simulation using numerical integration.

## Samples
- `SAMPLES/sample1.py`: Projectile motion under gravity with air drag.
- `SAMPLES/sample2.py`: Spring-mass-damper system using Euler integration.
- `SAMPLES/sample3.py`: 2D particle collision simulation with boundary bouncing.

## Running
```bash
python python-1000-snippets/0298-Physics-Simulation/SAMPLES/sample1.py
python python-1000-snippets/0298-Physics-Simulation/SAMPLES/sample2.py
python python-1000-snippets/0298-Physics-Simulation/SAMPLES/sample3.py
```

## Notes
- These examples use simple explicit time integration and are intended for learning.
- For more accurate physics, consider use of symplectic integrators or dedicated physics engines.
