# String Methods Cheatsheet

## Case conversions
- `upper()`, `lower()`, `title()`, `capitalize()`

## Trimming
- `strip()`, `lstrip()`, `rstrip()`

## Searching and replacing
- `find(sub)`, `replace(old, new)`

## Splitting and joining
- `split()`, `splitlines()`, `join()`

## Others
- `startswith()`, `endswith()`, `count()`

## Tips
- Strings are immutable; methods return new strings.
- Use `in` for membership testing: `'a' in s`

## Running samples
1. `python3 -m venv .venv && source .venv/bin/activate`
2. `python SAMPLES/sample1.py`
3. `python SAMPLES/sample2.py` (simulated input)
4. `python SAMPLES/sample3.py`
