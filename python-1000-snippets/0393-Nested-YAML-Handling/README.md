
# Nested YAML Handling

## Description
Demonstrates parsing and emitting YAML with nested structures using `pyyaml`.

## Files
- `SAMPLES/sample1.py` — Parse YAML from a string.
- `SAMPLES/sample2.py` — Parse YAML from a file.
- `SAMPLES/sample3.py` — Dump Python objects to YAML.

## Usage
```bash
python SAMPLES/sample1.py
python SAMPLES/sample2.py
python SAMPLES/sample3.py
```
