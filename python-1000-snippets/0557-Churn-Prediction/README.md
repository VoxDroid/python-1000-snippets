# Churn Prediction

## Description
Predicts churn using basic rules and risk aggregation.

## Samples
- `SAMPLES/sample1.py`: rule-based churn prediction.
- `SAMPLES/sample2.py`: churn risk fraction from predicted labels.
- `SAMPLES/sample3.py`: persist churn report to `temp/0557_churn.txt`.

## Output
- churn label and risk metric.
- report in temp file.
