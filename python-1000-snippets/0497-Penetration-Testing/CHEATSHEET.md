# 0497-Penetration-Testing Cheatsheet

## Quick Start
```bash
python3 python-1000-snippets/0497-Penetration-Testing/SAMPLES/sample1.py
```

## Notes
- `sample1.py`: Quick check for standard service ports.
- `sample2.py`: scan numeric port range for open ports.
- `sample3.py`: store scan results to `temp/port_scan_results.txt`.

## Tips
- Use small timeout values to prevent long waits.
- Log scan metadata (timestamp, host) in `temp/`.
