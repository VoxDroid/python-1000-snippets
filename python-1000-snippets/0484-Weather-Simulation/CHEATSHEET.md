# 0484-Weather-Simulation Cheatsheet

## Quick Start
Run a sample:
```bash
python3 python-1000-snippets/0484-Weather-Simulation/SAMPLES/sample1.py
```

## Notes
- `sample1.py` simulates temperature series with small noise and trend.
- `sample2.py` simulates pressure and humidity variations.
- `sample3.py` computes wind chill for selected points.

## Tips
- Extend to wind/precipitation arrays and store in CSV for analysis.
- Add daily/cyclical patterns to make the model more realistic.
