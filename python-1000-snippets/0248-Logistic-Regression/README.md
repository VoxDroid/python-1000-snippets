# Logistic Regression

## Description
This snippet demonstrates training and evaluating a binary logistic regression classifier using `scikit-learn`.

## Code
In the `SAMPLES/` folder you will find three examples:

- `sample1.py` — train a logistic regression model and print test accuracy.
- `sample2.py` — show predicted class probabilities for sample inputs.
- `sample3.py` — compute a confusion matrix for model evaluation.

Run any of them with:

```bash
python python-1000-snippets/0248-Logistic-Regression/SAMPLES/sample1.py
```

## Output
Each sample prints accuracy or classification metrics based on synthetic data.

## Explanation
- **Logistic Regression**: Models probability of a binary outcome.
- **Logic**: Generates synthetic binary data, trains the model, and evaluates performance.
- **Best Practice**: Use regularization, evaluate with confusion matrix/ROC, and scale inputs.
