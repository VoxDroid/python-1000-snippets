# Customer Segmentation

## Description
Segments customers with clustering and rule-based categorization.

## Samples
- `SAMPLES/sample1.py`: simple K-means style clustering (pure Python) for segment labels.
- `SAMPLES/sample2.py`: business rule segmentation by purchase frequency and value.
- `SAMPLES/sample3.py`: writes output to `temp/0556_segmentation.txt`.

## Output
- labels and centroids plus segment groups.
- final temp file with assignments.
