# 0481-Real-Time-Rendering Cheatsheet

## Quick Start
Run a sample:
```bash
python3 python-1000-snippets/0481-Real-Time-Rendering/SAMPLES/sample1.py
```

## Notes
- This snippet simulates a render loop without a graphics library.
- `sample2.py` writes frame snapshots to `temp/realtime_rendering_frames.txt`.
- `sample3.py` measures frame timing to compute approximate FPS.

## Tips
- Adjust the `width`, `height`, or `frames` values to experiment with performance.
- Use the output file from `sample2.py` to visualize frame changes over time.
