# Deep Learning Pipeline

## Description
This snippet demonstrates a simple deep learning pipeline using scikit-learn's `MLPClassifier`.

## Requirements
- Python 3.8+
- `scikit-learn` (`pip install scikit-learn`)
- `joblib` (`pip install joblib`)

## Samples
- `SAMPLES/sample1.py`: Train an MLP pipeline on the digits dataset.
- `SAMPLES/sample2.py`: Save and load an MLP model using joblib.
- `SAMPLES/sample3.py`: Inspect the trained MLP weight matrices.

## Running
```bash
python python-1000-snippets/0454-Deep-Learning-Pipeline/SAMPLES/sample1.py
python python-1000-snippets/0454-Deep-Learning-Pipeline/SAMPLES/sample2.py
python python-1000-snippets/0454-Deep-Learning-Pipeline/SAMPLES/sample3.py
```
