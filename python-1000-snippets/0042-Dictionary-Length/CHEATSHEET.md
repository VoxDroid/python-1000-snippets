# Dictionary Length Cheatsheet

## Using `len()` on dictionaries
- Returns number of keys.
- Equivalent to `len(dict.keys())`.

## Tips
- Empty dict returns 0.
- For nested dicts, you may need recursive counting.

## Example
```python
if len(my_dict) == 0:
    print('no entries')
```
