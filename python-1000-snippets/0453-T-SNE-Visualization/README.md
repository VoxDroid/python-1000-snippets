# t-SNE Visualization

## Description
This snippet demonstrates using t-SNE for dimensionality reduction and embedding of high-dimensional data.

## Requirements
- Python 3.8+
- `scikit-learn` (`pip install scikit-learn`)
- `pandas` (`pip install pandas`)

## Samples
- `SAMPLES/sample1.py`: Compute and print t-SNE embedding coordinates.
- `SAMPLES/sample2.py`: Save t-SNE embedding to `/temp/tsne_iris.csv`.
- `SAMPLES/sample3.py`: Compare embeddings across different perplexity values.

## Running
```bash
python python-1000-snippets/0453-T-SNE-Visualization/SAMPLES/sample1.py
python python-1000-snippets/0453-T-SNE-Visualization/SAMPLES/sample2.py
python python-1000-snippets/0453-T-SNE-Visualization/SAMPLES/sample3.py
```
