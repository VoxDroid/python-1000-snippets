# Virtual Reality

## Description
This snippet demonstrates simple VR-style rendering techniques using basic 3D projections.

## Samples
- `SAMPLES/sample1.py`: Render a stereo pair (left/right eye) of a cube and save images.
- `SAMPLES/sample2.py`: Simulate head rotation by projecting a 3D point cloud at different angles.
- `SAMPLES/sample3.py`: Combine a stereo pair into a red-cyan anaglyph image.

## Running
```bash
python python-1000-snippets/0296-Virtual-Reality/SAMPLES/sample1.py
python python-1000-snippets/0296-Virtual-Reality/SAMPLES/sample2.py
python python-1000-snippets/0296-Virtual-Reality/SAMPLES/sample3.py
```

## Notes
- Generated images are stored in `/tmp` by default.
- For real VR apps, use 3D graphics engines or OpenGL for interactive rendering.
