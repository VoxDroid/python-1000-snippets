# Transfer Learning

## Description
This snippet demonstrates a basic transfer learning workflow by training a base model, saving it, then loading and fine-tuning it on new data.

## Requirements
- Python 3.8+
- `scikit-learn` (`pip install scikit-learn`)
- `joblib` (`pip install joblib`)

## Samples
- `SAMPLES/sample1.py`: Train a base model and save it.
- `SAMPLES/sample2.py`: Load the model and fine-tune on new data.
- `SAMPLES/sample3.py`: Compare pre- and post-fine-tuning performance.

## Running
```bash
python python-1000-snippets/0455-Transfer-Learning/SAMPLES/sample1.py
python python-1000-snippets/0455-Transfer-Learning/SAMPLES/sample2.py
python python-1000-snippets/0455-Transfer-Learning/SAMPLES/sample3.py
```
